dockerApplicationServer
=======================

dockerApplicationServer

Starta med:
  vagrant up --no-parallel --provider docker

Förutsättningar:
 VirtualBox
 Vagrant från vagrantup.com

Surfa därefter till:
  http://127.0.0.1:28153/go för go:

och:
 http://127.0.0.1:58080 för doodleshop i produktion

Övriga portmappings:
 28080 autosmall
 38080 autolarge
 48080 man
 58080 prod

Så får du upp en pipeline
